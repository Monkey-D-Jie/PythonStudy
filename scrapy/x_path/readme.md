<h1>XPath表达式基础语法说明</h1>

<h2>1.逐层提取</h2>

- 语法：就一个“/”
- 示例：，html/head，表示获取到html下的head标签

<h2>2.提取标签下的文本</h2>

- 语法：text()
- 示例：/html/head/title/text()，表示获取到title下的标题内容

<h2>3.提取指定名称的所有标签</h2>

- 语法：//标签名
- 示例：//div，表示提取所有的div标签

<h2>4.提取指定属性名称的标签</h2>

- 语法：//标签名[@属性名='属性值']
- 示例：//div[@class='tools'],表示提取所有div标签下的属性名为”class“,
属性值为”tools“的标签的内容
